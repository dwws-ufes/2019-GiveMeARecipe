## Deploy

**Requirements:**
* docker
* docker-compose

**How to run:**
1. Navigate to GiveMeARecipeAspNetCore directory.
2. Run on your terminal:
    ```
    $ docker-compose up
    ```
3. Wait for the process to complete. The last line will be "Content root path: /app".
4. Go to your browser and access http://localhost:5000.
